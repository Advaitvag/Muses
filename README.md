## Dependencies

This project uses the `metadata_god` package, which requires Rust to be installed on your system.
You can install Rust by following the instructions on the [official Rust website](https://www.rust-lang.org/tools/install).

On Linux, this application uses `mpv` for audio playback. Please ensure `mpv` is installed on your system. You can usually install it via your distribution's package manager (e.g., `sudo apt install mpv` on Debian/Ubuntu, `sudo dnf install mpv` on Fedora, `sudo pacman -S mpv` on Arch Linux).
